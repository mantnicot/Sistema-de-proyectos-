Coloque aquí los logos institucionales para PDF y Word.

## Archivos requeridos

| Archivo | Descripción |
|---------|-------------|
| `universidad.png` | Logo Universidad Distrital (escudo) |
| `oati.png` | Logo OATI |

También se aceptan `.jpg`, `.jpeg` o `.svg` con el mismo nombre.

**Nota:** Si `oati.png` es un archivo SVG, el sistema lo convierte automáticamente al exportar.

## Recomendaciones

- Formato PNG con fondo transparente
- Resolución mínima 300×300 px
- Nombre exacto: `universidad.png` y `oati.png`

Tras colocar los archivos, reinicie el backend para que aparezcan en los documentos exportados.
